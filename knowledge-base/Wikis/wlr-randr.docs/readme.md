# wlr-randr

> **Warning**: this project has [moved to gitlab.freedesktop.org].

Utility to manage outputs of a Wayland compositor.

## Building

Install dependencies:

* meson (compile-time dependency)
* wayland

Then run:

    meson build
    ninja -C build
    build/wlr-randr
