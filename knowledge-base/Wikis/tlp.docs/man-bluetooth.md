# NAME

bluetooth - enable/disable internal bluetooth device

# SYNOPSIS

**bluetooth \[*command*\]**

# COMMANDS

**on**
Switch device on.

**off**
Switch device off.

**toggle**
Toggle device state.

**cycle**
Toggle the device twice (to return to the original state).

\<none\>
Show device state.

**--version**
Print TLP version.
